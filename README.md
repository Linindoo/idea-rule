# idea-rule

基于IDEA平台的常用正则表达式插件

### 安装

IDEA应用商店中搜索"**any-rule**".

### 使用
**方式1:**

右键选择**Any Rule** 打开正则列表

![](http://www.52zhoujia.cn/upload/2020/04/o8ks49pfnmhisq30bmpt9obpb3.gif)
**方式2:**

按**alt + a**打开正则列表

![](http://www.52zhoujia.cn/upload/2020/04/o8ks49pfnmhisq30bmpt9obpb3.gif)
